NAME

startapp - show status of apps on target

SYNOPSIS

startapp APP_NAME [TARGET_IP]

DESCRIPTION

If the application APP_NAME is installed on the target at TARGET_IP then the app will be started. If the APP_NAME is given as "*" (including the quotes) then all apps on the target will be started.

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
