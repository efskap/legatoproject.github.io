NAME

settime - set the time on target

SYNOPSIS

settime [TARGET_IP]

DESCRIPTION

Sets the target time to be the same as the host time. The time is synchronized on the basis of UTC time but the time displayed by date or the time stamps in syslog will use timezone information if set by settz.

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP

NOTES

This tool creates a master ssh socket named root@_legato_tools in .ssh/ so that it can open subsequent connections quickly. The socket is deleted if the script ends normally.
