The update tool can be used to install, update, and remove software on the target device.

It mainly operates on "update packs", which are files containing install, update, and/or removal instructions and any files needed to carry out those instructions.

Update packs are created on the development host using the update-pack tool.

Usage

update [FILE_NAME]
update -r APP_NAME
update –remove APP_NAME
update –help

When run without any command-line arguments, or when run with FILE_NAME set to "-", takes an update pack file through standard in and forwards it to the Update Daemon for processing.

update
update -

When run with an update pack's file path as its only argument, opens the update pack and feeds it to the Update Daemon for processing.

When run with the –remove or -r option, removes the app specified in the APP_NAME argument.

update /tmp/myUpdatePack.update
Warning
Be careful not to accidentally remove system services apps that you might need (e.g., modemService app).

When run with the –help option, prints help text to standard out and exits.
