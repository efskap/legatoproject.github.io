This prototype sample demonstrates a CF3 GPIO app using the GPIO API Service. It uses adef executables, bindings, and processes sections, and cdef requires, cflag, and sources sections.

Run this to install the sample app:

$ cd apps/sample/gpioCf3Demo
$ make wp85
$ instapp gpioCf3Demo.wp85

If you're using an AR7 or WP7 module in your target, substitute ar7 or wp7 for wp85.
