NAME

rmapp - remove app from target

SYNOPSIS

rmapp APP_NAME [TARGET_IP]

DESCRIPTION

Remove the app given by APP_NAME from the IP address of the target, TARGET_IP If APP_NAME is given as "*" (including the quotes) then remove all apps

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
