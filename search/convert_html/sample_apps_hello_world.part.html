This sample app demonstrates target and host are communicating using a helloWorld app. It has an .adef with executables and processes sections.

Run this to install the sample app:

$ cd apps/sample/helloWorld
$ make wp85
$ instapp helloWorld.wp85

If you're using an AR7 or WP7 module in your target, substitute ar7 or wp7 for wp85.
