NAME

simu - helper script to use the simulation environement

SYNOPSIS

simu start
simu stop
simu shell | ssh | sh
simu push [LOCAL] [REMOTE]
simu pull [LOCAL] [REMOTE]
simu instlegato
simu instapp [APP_PATH]

DESCRIPTION

simu start

  • Start QEmu.

simu stop

  • Stop QEmu.

simu shell simu sh simu ssh

  • Get into a shell on the target.

simu push [LOCAL] [REMOTE] simu pull [LOCAL] [REMOTE]

  • Transmit a file or directory to/from the target.

simu instlegato

  • Install Legato on the virtual target.

simu instapp [APP_PATH]

  • Install application from APP_PATH on the virtual target.
