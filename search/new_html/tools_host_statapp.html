NAME

statapp - show status of apps on target

SYNOPSIS

statapp [TARGET_IP]

DESCRIPTION

The target at TARGET_IP will be queried to find out the status of all installed applications and a list of all running and stopped apps will be returned

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
