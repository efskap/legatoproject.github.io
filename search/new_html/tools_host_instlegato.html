NAME

instlegato - install legato on target

SYNOPSIS

instlegato BUILD_DIR [TARGET_IP]

DESCRIPTION

Once Legato has been built, instlegato can be used to install it on the target at the IP address specified by TARGET_IP. BUILD_DIR is the path to the directory to which Legato was built. If the target Legato directory is in the normal build location, i.e. Inside $LEGATO_ROOT/build, then you only need to specify the target platform, e.g. ar7, otherwise a full path is required.

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
