These are the high-level steps to prepare the dev kit board for Legato:

  • Assemble Dev Kit with Legato-ready module.
  • configure Board Switches to default settings.
  • attach Dev Kit Cables to board:
    • UART2
    • Ethernet or micro-USB
    • power supply.
  • attach cables to dev host.
  • power up board.
