This sample app demonstrates SMS messaging. It uses .adef bindings and related @ .cdef requires and sources sections.

Before you install this sample app, register your SIM network. See SIM.

Run this to install the sample app: 

$ cd apps/sample/sms
$ make wp7
$ instapp sms

If you're using an AR7 module in your device, substitute ar7 for wp7.
