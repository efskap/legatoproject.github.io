You need to Configure IP Address before you can install Legato on a target.

We use a serial communication program like Minicom to access the target through the UART port to for console communications. Ethernet and Micro-USB ports are used for development. See Dev Kit Cables for more info.

If you're using Minicom, you access the config and initialize menus through CTRL+A Z.

Run as root user, so no password is required.

Once the host and target are communicating, you can complete the target setup.

Note
Use the setNet tool to set your MAC address so you don't have to go through ifconfig every time you reboot the target.
